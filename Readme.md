# Privacy-Preserving Clinical Decision Support System (CDSS)

This project implements a local, privacy-preserving Clinical Decision Support System using Retrieval-Augmented Generation (RAG) with vector databases and offline large language models.

## Tech Stack
- Python
- Ollama (LLaMA3)
- PostgreSQL + pgvector
- FastAPI
- Streamlit
- n8n
- Docker
